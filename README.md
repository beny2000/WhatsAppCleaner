# WhatsAppCleaner
This Python script sorts and cleans WhatsApp groupchat message data of mixed sender into each sender and their respective messages for use in classification algorithms. 

data_format.txt
```
SENDER_NAME: MESSAGE \n
SENDER_NAME: MESSAGE \n
SENDER_NAME: MESSAGE \n
SENDER_NAME: MESSAGE \n
SENDER_NAME: MESSAGE \n
SENDER_NAME: MESSAGE \n
.
.
.
```
